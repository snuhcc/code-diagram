## Environment Setup
```bash
# Clone repository
git clone <repository URL>
cd <repository dir>

# Create Conda environment
conda create --name <env-name> python=3.13
conda activate <env-name>

# Install dependencies
pip install -r requirements.txt
```

## Frontend
```bash
cd frontend
pnpm install
pnpm dev

http://localhost:3000/
```
